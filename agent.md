## UI 视觉与交互规范 (Visual Identity)

### 风格定义
*   **关键词**：极简 (Minimalist)、禅意 (Zen)、安全感 (Safe)。
*   **核心理念**：减少视觉噪音，提供一个可以“深呼吸”的数字空间。

### 调色盘 (Color Tokens)
*   **主色调 A**：`#B2AC88` (鼠尾草绿) —— 用于背景、主要按钮，象征和平与缓解焦虑。
*   **主色调 B**：`#E6D5B8` (暖沙色) —— 用于辅助背景、卡片容器，营造温馨感。
*   **辅助色**：`#5F7ADB` (柔和靛青) —— 用于 AI 仲裁结果、深度建议，代表理智。
*   **禁忌色**：严禁使用亮红色 (#FF0000) 或高饱和度荧光色。

### UI 进阶实现建议 (Implementation Tips)
*   **留白 (Negative Space)**：保持极高的留白率。界面不要堆叠信息，每个组件之间留出足够的“呼吸感”。
*   **阴影与质感**：使用极其轻微、弥散度高的柔和阴影 (Soft Shadows)，让 UI 组件看起来像轻盈地漂浮在暖沙色背景上。
*   **动态背景 (Motion Logic)**：AI 在处理或思考时，背景色应在鼠尾草绿与暖沙色之间进行极慢的线性渐变切换，模拟人类的深呼吸频率。
*   **形状 (Shapes)**：使用超大圆角 (e.g., `border-radius: 32px`)，消除任何尖锐边缘带来的心理防御。
